Параметр Описание Примеры
name

Имя станицы Wikivoyage, с которой будут загружены объекты. Также можно указать несколько страниц через запятую.

Если указана одна страница, то координаты центра карты, масштаб и Wikidata ID отображаемой области будут загружены автоматически из параметров шаблона {{monument-title}}. В противном случае, предпочтительно вручную явно задать значения параметров lat, long, zoom и districtid.

  • Природные памятники: ?name=Природные_памятники_России/Алтай
  • Культурное наследие: ?name=Культурное_наследие_России/Красноярский_край/Красноярск
  • Несколько страниц через запятую: ?name=Культурное_наследие_России/Новосибирская_область/Новосибирск,Культурное_наследие_России/Новосибирская_область/Районы_Новосибирской_области
name-prefix

Префикс страниц, с которых будут загружены страницы.

При определении, необходимо ли загружать данные со страницы, используется простое сравнение - начинается ли название страницы с указанного префикса. В определенных случаях, это может привести к неожиданным результатам, например, если указать в качестве префикса "Культурное_наследие_России/Томская_область/Томск", помимо Томска будут загружены объекты со страницы "Культурное_наследие_России/Томская_область/Томский_район". Чтобы избежать такого поведения, следует использовать параметр name-prefix-parts, описаный ниже.

  • Все страницы культурного наследия Томской области (начинающиеся с "Культурное_наследие_России/Томская_область"): ?name-prefix=Культурное_наследие_России/Томская_область
  • Все страницы Санкт-Петербурга (осторожно: загрузка карты может занять много времени и ресурсов): ?name-prefix=Культурное_наследие_России/Санкт-Петербург
  • Все страницы Москвы (осторожно: загрузка карты может занять много времени и ресурсов): ?name-prefix=Культурное_наследие_России/Москва
name-prefix-parts

Префикс страниц, разбитых на несколько частей.

При определении, необходимо ли загружать данные со страницы, используется логика разбиения страниц на части, а именно: каждая часть начинается с открывающейся скобки.

  • Все страницы культурного наследия Томска ("Культурное_наследие_России/Томская_область/Томск (часть 1)", "Культурное_наследие_России/Томская_область/Томск (часть 2), и т.д."): ?name-prefix-parts=Культурное_наследие_России/Томская_область/Томск
lat Широта центра карты, в десятичном формате
  • ?name=Культурное_наследие_России/Бурятия/Кяхта&lat=50.32360
long Долгота центра карты, в десятичном формате
  • ?name=Культурное_наследие_России/Бурятия/Кяхта&long=106.40396
zoom Масштаб, от 2 (наиболее общий обзор) до 17 (наибольшее увеличение)
  • ?name=Культурное_наследие_России/Бурятия/Кяхта&zoom=10
sequential

Способ загрузки страниц в случае, если их несколько.

Если параметр задан и равен true, тогда страницы будут загружаться последовательно, одна за другой, таким образом минимизируя загрузку серверов Wikimedia.

Если параметр не задан, тогда страницы загружаются параллельно, минимизируя время загрузки карты.

  • Параллельная загрузка: ?name=Культурное_наследие_России/Томская_область/Томск_(часть_1),Культурное_наследие_России/Томская_область/Томск_(часть_2),Культурное_наследие_России/Томская_область/Томск_(часть_3),Культурное_наследие_России/Томская_область/Томск_(часть_4)
  • Последовательная загрузка: ?name=Культурное_наследие_России/Томская_область/Томск_(часть_1),Культурное_наследие_России/Томская_область/Томск_(часть_2),Культурное_наследие_России/Томская_область/Томск_(часть_3),Культурное_наследие_России/Томская_область/Томск_(часть_4)&sequential=true
hide-boundaries

Если параметр задан, то границы объектов (например, заповедников, национальных парков) не будут отображаться.

По-умолчанию, границы отображаются.

  • Границы отображаются: ?name=Природные_памятники_России/Алтай
  • Границы не отображаются: ?name=Природные_памятники_России/Алтай&hide-boundaries=true
hide-markers

Если параметр задан, то значки объектов не будут отображаться.

Может пригодиться в случае, если хочется отображать только границы объектов (например, заповедников, национальных парков).

По-умолчанию, значки отображаются.

  • Значки отображаются: ?name=Природные_памятники_России/Алтай
  • Значки не отображаются: ?name=Природные_памятники_России/Алтай&hide-markers=true
no-group-markers

Если параметр задан, то значки объектов не будут группироваться в кластера.

По-умолчанию, значки объектов группируются для ускорения работы карты и более наглядного отображения массовых скоплений объектов.

  • Значки группируются: ?name=Культурное_наследие_России/Бурятия/Кяхта
  • Значки не группируются: ?name=Культурное_наследие_России/Бурятия/Кяхта&no-group-markers=true
districtid

Идентификатор Викиданных области/региона, отображаемой на карте

Если параметр задан, и из Викиданных возможно загрузить границы области, тогда вся остальная карта, кроме этой области, будет затенена.

  • Культурное наследие Новосибирска с выделенным Ленинским районом: ?name=Культурное_наследие_России/Новосибирская_область/Новосибирск&districtid=Q4258377
hide-district-boundaries

Если параметр задан, то затенение карты по параметру districtid будет отключено.

  • С затенением: ?name=Природные_памятники_России/Алтай
  • Без затенения: ?name=Природные_памятники_России/Алтай&hide-district-boundaries=true
disable-auto-fit

По-умолчанию, если не заданы координаты центра карты (lat и long) и уровень масштабирования (zoom), карта автоматически отцентруется и отмасштабируется по значкам.

Эта опция отключает такое поведение, карта будет отпозиционирована по фиксированным координатам.

  • С автоматическим определением центра карты и масштаба: ?name-prefix-parts=Культурное_наследие_России/Томская_область/Томск
  • Без автоматического определения центра карты и масштаба: ?name-prefix-parts=Культурное_наследие_России/Томская_область/Томск&disable-auto-fit=true
show-address

По-умолчанию, для культурного наследния, во всплывающем окне объекта отображается его адрес. Для природных памятников же адрес по-умолчанию не отображается.

Эта опция вручную контролирует показ адресов во всплывающих окнах объектов.

  • С адресами: ?name=Культурное_наследие_России/Красноярский_край/Красноярск&show-address=true
  • Без адресов: ?name=Культурное_наследие_России/Красноярский_край/Красноярск&show-address=false
highlight-knid

Подсветить объект, имеющий указанный knid. Объект будет обведен в красный круг, карта будет отцентрована на объекте (если координаты не заданы явно), масштаб будет выбран автоматически (если не задан явно).

Опция плохо совместима с группировкой маркеров, поэтому ее рекомендуется отключать при помощи no-group-markers=true.

  • Красноярск c подсвеченной часовней Параскевы Пятницы: ?name=Культурное_наследие_России/Красноярский_край/Красноярск&no-group-markers=true&highlight-knid=2410008000
auto-boundary-markers

Экспериментальная опция, к использованию не рекомендуется.

Автоматически добавляет значек объекта для каждого полигона.
